# Finance-GPT

Finance-GPT is a ChatGPT-like AI chatbot specialized in answering ALL questions related to finance.

### Features
- Personal finance advice
- Budgeting guidance
- Investment suggestions
- Stock market explanations
- Crypto and trading insights
- Loan, EMI, credit score guidance
- Financial Python code generation

### How to Run

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Set your OpenAI API key in the `.env` file

3. Run the application:
   ```bash
   streamlit run app.py
   ```

### Usage

Simply type any finance-related question in the chat interface and get expert financial advice!
