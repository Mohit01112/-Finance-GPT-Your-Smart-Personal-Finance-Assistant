# Finance-GPT: Your Smart Personal Finance Assistant

Finance-GPT is a Flask-based web application powered by the Groq API that specializes in answering ALL questions related to finance. It acts as your personal financial advisor, providing expert guidance on various financial topics.

## Features
- Personal finance advice
- Budgeting guidance
- Investment suggestions
- Stock market explanations
- Crypto and trading insights
- Loan, EMI, credit score guidance
- Financial planning assistance
- Real-time financial data interpretation

## How to Run

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Set your Groq API key in the `.env` file:
   ```
   GROQ_API_KEY=your_api_key_here
   ```

3. Run the application:
   ```bash
   python app.py
   ```

4. Open your browser and go to `http://localhost:5000`

## Usage

Simply type any finance-related question in the chat interface and get expert financial advice!

## API Provider

This application uses the Groq API with the llama-3.1-8b-instant model for fast and efficient financial assistance.
